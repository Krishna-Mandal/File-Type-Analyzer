Dealing with currency

You have a class BankAccount, which has three methods: addDollars(), addEuro(), and addRubles(). All these methods change your balance:

class BankAccount {
    var balance = 0L

    fun addMoney(amount: Int, action: String) {
        when (action) {
            "1" -> addDollars(amount)
            "2" -> addEuro(amount)
            else -> addRubles(amount)
        }
    }

    fun addDollar() {
        balance += 75
    }
    fun addEuro() {
        balance += 84
    }
    fun addRuble() {
        balance += 1
    }
}

The program reads the number of threads and creates the threads that will change your balance.

Use block synchronization to ensure the correct work of this program.
