Thread-safe account

You have a class named Account. It has a long field storing the current balance. The account has two unimplemented methods:

  • the deposit method adds an amount to the balance;
  • the withdraw method subtracts an amount from the balance if the account has enough money. The method must return true if the operation is successful, otherwise, it returns false.

Implement the given methods. The class should work correctly with multiple threads. The testing system will create threads and call the methods on the same instance of Account.
